Variables

How to use variables and what they can do.

  
//Num is an number/integer
//Msg is a message/string
//Yon is true/false/possibly
Num i = 1
Msg o = "Hello "
Yon m = true

//Better Example Below
    
// Define a custom data type
datatype Person {
  msg name;
  num age;
  yon isStudent;
}

// Create a variable of type Person
Person person1;
person1.name = "John";
person1.age = 25;
person1.isStudent = true;

// Access and modify the properties of the Person object
print(person1.name); // Output: John
person1.age = 26;
print(person1.age); // Output: 26
